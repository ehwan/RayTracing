# RayTracing
ray tracing engine

## Dependencies
This project is using [Eigen3](https://eigen.tuxfamily.org/) as linear algebra, [SFML](https://github.com/SFML/SFML) for window management and [RTree](https://github.com/ehwan/RTree) for spatial indexing.
